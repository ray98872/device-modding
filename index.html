Tinkering with hardware, exploring the homebrew community.

I enjoy tinkering with hardware and exploring the homebrew community. It started back in 2011 when I was 11 years old, discovering XDA Developers to install custom ROMs on my Android phones. Since then, by following technical guides and community documentation, I've successfully soft-modded and jailbroken several generations of devices and consoles to run custom firmware and homebrew applications.

It's a fun way to learn about how different systems handle boot sequences and file management.

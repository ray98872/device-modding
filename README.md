# Device Modding & Hardware Tinkering

A hobby write-up: soft-mods, jailbreaks and homebrew across mobile, PlayStation, Nintendo and Xbox — started in 2011 on XDA Developers. This repo is the standalone showcase page.

**Live page:** https://ray98872.github.io/device-modding/

## Covered

- **Mobile & Tablets** — custom Android ROMs on HTC Desire / HTC Sensation (XDA), iPad 3 jailbreak (2013)
- **PlayStation** — FreeMcBoot on PS2 (007: Agent Under Fire disc-swap), CFW on Fat PS3 / Hybrid on Slim, tethered PPPwn on PS4
- **Nintendo** — R4 on 2DS XL, LetterBomb on the original Wii, Aroma on Wii U
- **Xbox** — classic JTAG hardware mod on the Xbox 360
